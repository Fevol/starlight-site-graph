This package also exports all the components that are used to generate the new sidebar.
So, if you want to use these components in different places, you can do so by importing them from the package.

<Tabs>
	<TabItem label='MDX' icon='mdx'>
		```mdx
		// src/content/docs/my-page.mdx
		---
		title: My Page
		---

		import { PageGraph } from 'starlight-site-graph/components';

		<PageGraph />
		```
	</TabItem>

	<TabItem label='Markdoc' icon='mdoc'>
		```mdx
		// src/content/docs/my-page.mdx
		---
		title: My Page
		---

		import { PageGraph } from 'starlight-site-graph/components';

		<PageGraph />
		```
	</TabItem>

	<TabItem label='Astro' icon='astro'>
		```astro
		// my-component.astro
		---
		import { PageGraph } from 'starlight-site-graph/components';
		---

		<PageGraph />
		```
	</TabItem>
</Tabs>
