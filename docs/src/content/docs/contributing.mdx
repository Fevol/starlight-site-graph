---
title: Contributing
description: Help improve the Starlight Site Graph plugin by contributing to the project.
---

If you notice any issue with the plugin, or if you have a feature request, please do not
hesitate to open an issue on the [GitHub repository](https://github.com/fevol/starlight-site-graph/issues/new).

Feel free to contribute to the project by opening a pull request.
The project is open to any kind of contribution, be it a bug fix, a new feature, improving some lackluster documentation,
or correcting one of the (many) typos lurking around in the documentation.
