## Quick Start

<Steps>
1.  Setup a Starlight Site: https://starlight.astro.build/getting-started/

2.  Install the `starlight-site-graph` plugin:
    <PackageManagers pkg="starlight-site-graph" />

3.  Update the `integrations.starlight.plugins` object in `astro.config.mjs` to include the `starlight-site-graph` plugin:

    ```ts ins={4, 10}
    // astro.config.mjs
    import { defineConfig } from "astro/config";
    import starlight from "@astrojs/starlight";
    import starlightSiteGraph from "starlight-site-graph";

    // https://astro.build/config
    export default defineConfig({
        integrations: [
            starlight({
                plugins: [starlightSiteGraph()],
            }),
        ],
    });
    ```
