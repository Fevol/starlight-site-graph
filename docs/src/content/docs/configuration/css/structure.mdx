This page contains information about the CSS structure of the plugin (i.e. how the components exist within the DOM).
An example style override for the components could look something like:

```css
// src/styles/global.css
.graph-container {
    border: 1px solid red;
}
```

## Graph
### Graph Container
- `(.right-sidebar-panel .slsg-graph-panel)`: Holds the entire graph component. Classes passed by component props.
  - `h2`: Graph title.
  - `.slsg-graph-component .slsg-graph-skeleton`: Only visible while the graph is loading, ensures visual consistency.
  - `.slsg-graph-component`: Holds all graph-related elements.
    - `.slsg-graph-container (.slsg-is-fullscreen)`: Holds the graph canvas, fullscreen class is applied when maximized.
      - `.slsg-graph-action-container`: Holds the graph action icons.
        - `button.slsg-graph-action-button`: A graph action icon.
            - `svg`: Action icon.
      - `canvas`: Where the graph is actually drawn.
    - (`.slsg-background-blur`: Element is only added when the graph is maximized, blurs the site background.)

You can also style the performance stats when [debug](/starlight-site-graph/configuration/general#debug) is enabled:
- `#slsg-graph-stats`: Performance stats container.


### Context Menu
On right clicking a `.slsg-graph-action-button`, a context menu will appear:
- `body`: Document body
  - `slsg-menu-container`: Context menu container, positions the context menu.
    - `slsg-menu`: Context menu items list
      - `slsg-menu-item`: Context menu item
        - `slsg-menu-item-icon`: Context menu item icon
        - `slsg-menu-item-title`: Context menu item title

### Popup Menu
On activating the simulation settings (via the corresponding quick action), a popup menu will be created
inside the `.slsg-graph-action-container`:
- `.slsg-graph-container`: Graph container
  - `.slsg-graph-action-container`: Holds the graph action icons.
    - `button.slsg-graph-action-button`: A graph action icon.
    - **`.slsg-popup-menu`**: Popup menu container
      - `.slsg-popup-menu-content`: Popup menu item
        - `.slsg-value-slider`: Slider for adjusting configuration values
          - `.slsg-value-slider-text`: Slider label
          - `input[type="range"]`: Slider input

## Backlinks

- `(.right-sidebar-panel .slsg-backlinks-panel)`: Holds the backlinks component. Classes passed by component props.
  - `h2`: Backlinks title.
  - `ul .slsg-backlinks`: Backlinks list element
    - `li` Backlink item
      - `a .slsg-backlink`: Backlink anchor
        - `span`: Backlink text
