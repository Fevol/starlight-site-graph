Each page has a title, and so each node in the graph can have a label associated with it.
Here, you can find information on how you can customize the rendering of these labels, such as
their size, relative position and opacity (under different conditions).

:::note[Note]
If you are looking for information on how to _color_ these labels,
this can be done by changing its corresponding [CSS variables](/starlight-site-graph/configuration/css/variables#labels).
:::

## Label Rendering

The `renderLabels` option toggles the rendering of labels on the nodes of the graph. When labels are rendered, they display the name of the node.

<GraphExample id="render-labels-showcase"
	config={{
		graphConfig: {
			renderLabels: true,
		}
	}}
	baseConfig={overrideConfig(focusSingleNode, { graphConfig: { scale: 2.3 }})}
	sitemap={singleNodeSitemap} slug={"/node/"}
	buttons={{
		"Hide Labels": { config: { graphConfig: { renderLabels: false } } },
		"Render Labels": { config: { graphConfig: { renderLabels: true } } },
	}}
/>

## Label Font Size

The `labelFontSize` determines the size of the labels on the nodes of the graph.

<GraphExample id="label-font-size-showcase"
	config={{
		graphConfig: {
			labelFontSize: 12,
			renderLabels: true,
		}
	}}
	baseConfig={overrideConfig(focusSingleNode, { graphConfig: { scale: 2.3 }})}
	sitemap={singleNodeSitemap} slug={"/node/"}
	sliders={{
		"graphConfig.labelFontSize": {
				"title": "labelFontSize",
				"min": 1,
				"max": 50,
				"step": 1,
				"val": 12,
		}
	}}
/>


## Label Offset

The `labelOffset` determines the distance between the label and the node.

<GraphExample id="label-offset-showcase"
	config={{
		graphConfig: {
			labelOffset: 10,
			renderLabels: true,
		}
	}}
	baseConfig={overrideConfig(focusSingleNode, { graphConfig: { scale: 2.3 }})}
	sitemap={singleNodeSitemap} slug={"/node/"}
	sliders={{
		"graphConfig.labelOffset": {
			"title": "labelOffset",
			"min": -40,
			"max": 20,
			"step": 1,
			"val": 10,
		}
	}}
/>

## Label Scale (Hover)

The `labelHoverScale` determines the scale of the labels on the nodes of the graph when hovered over.

:::caution[Warning]
At larger scale levels, the labels will turn pixelated.
:::

<GraphExample id="label-hover-scale-showcase"
	config={{
		graphConfig: {
			labelHoverScale: 1.3,
			renderLabels: true,
		}
	}}
	baseConfig={overrideConfig(focusSingleNode, { graphConfig: { scale: 2.3 }})}
	sitemap={singleNodeSitemap} slug={"/node/"}
	sliders={{
		"graphConfig.labelHoverScale": {
			"title": "labelHoverScale",
			"min": 0.1,
			"max": 3,
			"step": 0.1,
			"val": 1.3,
		}
	}}
/>

## Label Offset (Hover)

The `labelHoverOffset` determines the distance between the label and the node when hovered over.
So when hovered, the label will be offset to the specified distance (it is _not_ additive to the base `labelOffset`).

<GraphExample id="label-hover-offset-showcase"
	config={{
		graphConfig: {
			labelHoverOffset: 14,
			renderLabels: true,
		}
	}}
	baseConfig={overrideConfig(focusSingleNode, { graphConfig: { scale: 2.3 }})}
	sitemap={singleNodeSitemap} slug={"/node/"}
	sliders={{
		"graphConfig.labelHoverOffset": {
			"title": "labelHoverOffset",
			"min": -40,
			"max": 20,
			"step": 1,
			"val": 14,
		}
	}}
/>

## Label Opacity Scale

The `labelOpacityScale` determines how opaque the labels on the nodes of the graph are, based on the zoom level.
A higher value will make the labels visible at lower zoom levels (when the user has zoomed out).

<GraphExample id="label-opacity-scale-showcase"
	config={{
		graphConfig: {
			labelOpacityScale: 1.3,
			renderLabels: true,
			enableZoom: true,
		}
	}}
  	baseConfig={frozenConfig} sitemap={largeRandomNodeSitemap} slug={"/node/"}
	sliders={{
		"graphConfig.labelOpacityScale": {
			"title": "labelOpacityScale",
			"min": 0,
			"max": 4,
			"step": 0.1,
			"val": 1.3,
		}
	}}
/>

The `labelHoverOpacity` determines how opaque the labels on the nodes of the graph are when hovered over.

## Label Opacity (Adjacent)

The `labelAdjacentOpacity` determines how opaque the labels on the nodes of the graph are when adjacent to the hovered node.
A higher value will make the labels more opaque when adjacent to the hovered node.

## Label Opacity (Muted)

The `labelMutedOpacity` determines how opaque the labels on the nodes of the graph are when the node is not adjacent or hovered over.
A higher value will make the labels more opaque when the node is not adjacent or hovered over.

<GraphExample id="label-opacity-muted-showcase"
	config={{
		graphConfig: {
			labelMutedOpacity: 0.00001,
			renderLabels: true,
		}
	}}
	baseConfig={overrideConfig(frozenConfig, { graphConfig: { repelForce: 0, colliderPadding: 10, scale: 1.3, depth: 5 }})}
	sitemap={twoNodeSitemap} slug={"/node/"}
	sliders={{
		"graphConfig.labelMutedOpacity": {
			"title": "labelMutedOpacity",
			"min": 0,
			"max": 1,
			"step": 0.1,
			"val": 0,
		}
	}}
/>
