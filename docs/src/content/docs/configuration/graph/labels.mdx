## Label Rendering

The `renderLabels` option toggles the rendering of labels on the nodes of the graph. When labels are rendered, they display the name of the node.

<GraphExample id="render-labels-showcase" config={{
	renderLabels: true,
}} baseConfig={{ ...focusSingleNode, scale: 2.3 }} sitemap={singleNodeSitemap} slug={"node/"}
	buttons={{
		"Hide Labels": { config: { renderLabels: false } },
		"Render Labels": { config: { renderLabels: true } },
	}}
/>

## Label Font Size

The `labelFontSize` determines the size of the labels on the nodes of the graph.

<GraphExample id="label-font-size-showcase" config={{
	labelFontSize: 12,
	renderLabels: true,
}} baseConfig={{ ...focusSingleNode, scale: 2.3 }} sitemap={singleNodeSitemap} slug={"node/"} sliders={{
	"labelFontSize": {
		"title": "labelFontSize",
		"min": 1,
		"max": 50,
		"step": 1,
		"val": 12,
	}
}} />


## Label Offset

The `labelOffset` determines the distance between the label and the node.

<GraphExample id="label-offset-showcase" config={{
	labelOffset: 10,
	renderLabels: true,
}} baseConfig={{ ...focusSingleNode, scale: 2.3 }} sitemap={singleNodeSitemap} slug={"node/"} sliders={{
	"labelOffset": {
		"title": "labelOffset",
		"min": -40,
		"max": 20,
		"step": 1,
		"val": 10,
	}
}} />

## Label Scale (Hover)

The `labelHoverScale` determines the scale of the labels on the nodes of the graph when hovered over.

:::caution[Warning]
At larger scale levels, the labels will turn pixelated.
:::

<GraphExample id="label-hover-scale-showcase" config={{
	labelHoverScale: 1.3,
	renderLabels: true,
}} baseConfig={{ ...focusSingleNode, scale: 2.3 }} sitemap={singleNodeSitemap} slug={"node/"} sliders={{
	"labelHoverScale": {
		"title": "labelHoverScale",
		"min": 0.1,
		"max": 3,
		"step": 0.1,
		"val": 1.3,
	}
}} />

## Label Offset (Hover)

The `labelHoverOffset` determines the distance between the label and the node when hovered over.
So when hovered, the label will be offset to the specified distance (it is _not_ additive to the base `labelOffset`).

<GraphExample id="label-hover-offset-showcase" config={{
	labelHoverOffset: 14,
	renderLabels: true,
}} baseConfig={{ ...focusSingleNode, scale: 2.3 }} sitemap={singleNodeSitemap} slug={"node/"} sliders={{
	"labelHoverOffset": {
		"title": "labelHoverOffset",
		"min": -40,
		"max": 20,
		"step": 1,
		"val": 14,
	}
}} />

## Label Opacity Scale

The `labelOpacityScale` determines how opaque the labels on the nodes of the graph are, based on the zoom level.
A higher value will make the labels visible at lower zoom levels (when the user has zoomed out).

<GraphExample id="label-opacity-scale-showcase" config={{
	labelOpacityScale: 1.3,
	renderLabels: true,
	enableZoom: true,
}} baseConfig={{ ...frozenConfig }} sitemap={largeRandomNodeSitemap} slug={"node/"} sliders={{
	"labelOpacityScale": {
		"title": "labelOpacityScale",
		"min": 0,
		"max": 4,
		"step": 0.1,
		"val": 1.3,
	}
}} />


## Label Opacity (Hover)

The `labelHoverOpacity` determines how opaque the labels on the nodes of the graph are when hovered over.
A higher value will make the labels more opaque when hovered over.

<GraphExample id="label-opacity-hover-showcase" config={{
	labelHoverOpacity: 1,
	renderLabels: true,
}} baseConfig={{ ...focusSingleNode, scale: 2.3 }} sitemap={singleNodeSitemap} slug={"node/"} sliders={{
	"labelHoverOpacity": {
		"title": "labelHoverOpacity",
		"min": 0,
		"max": 1,
		"step": 0.1,
		"val": 1,
	}
}} />


## Label Opacity (Adjacent)

The `labelAdjacentOpacity` determines how opaque the labels on the nodes of the graph are when adjacent to the hovered node.
A higher value will make the labels more opaque when adjacent to the hovered node.

<GraphExample id="label-opacity-adjacent-showcase" config={{

	renderLabels: true,
}} baseConfig={frozenConfig} sitemap={threeNodeSitemap} slug={"node/"} sliders={{

		"title": "labelAdjacentOpacity",
		"min": 0,
		"max": 1,
		"step": 0.1,
		"val": 1,
	}
}} />

## Label Opacity (Muted)

The `labelMutedOpacity` determines how opaque the labels on the nodes of the graph are when the node is not adjacent or hovered over.
A higher value will make the labels more opaque when the node is not adjacent or hovered over.

<GraphExample id="label-opacity-muted-showcase" config={{
	labelMutedOpacity: 0.00001,
	renderLabels: true,
}} baseConfig={{ ...frozenConfig, repelForce: 0, colliderPadding: 10, scale: 1.5 }} sitemap={twoNodeSitemap} slug={"node/"} sliders={{
	"labelMutedOpacity": {
		"title": "labelMutedOpacity",
		"min": 0,
		"max": 1,
		"step": 0.1,
		"val": 0,
	}
}} />
