The zoom logic of the graph can be altered to allow for a different initial zoom level, a more restrictive zoom extent,
snappier zoom animations, and changes in how the graph's arrows and links are change when zooming.

## Scale

The `scale` option determines the initial zoom level of the graph.

<GraphExample id="scale-showcase" config={{
	scale: 1.1,
}} baseConfig={{ ...focusSingleNode }} sitemap={singleNodeSitemap} slug={"node/"}
	sliders={{
		"scale": {
			"title": "scale",
			"min": 0.1,
			"max": 5,
			"step": 0.1,
			"val": 1.1,
		}
	}}
/>


## Minimum Zoom

The `minZoom` option determines the minimum zoom level of the graph, the lowest zoom extent that the user can zoom out to.
A higher value means that the user can zoom out less.

<GraphExample id="min-zoom-showcase" config={{
	minZoom: 0.05,
	enableZoom: true,
	enablePan: true,
}} baseConfig={{ ...focusSingleNode, scale: 1.1 }} sitemap={singleNodeSitemap} slug={"node/"}
	sliders={{
		"minZoom": {
			"title": "minZoom",
			"min": 0.1,
			"max": 5,
			"step": 0.1,
			"val": 0.05,
		}
	}}
/>


## Maximum Zoom

The `maxZoom` option determines the maximum zoom level of the graph, the highest zoom extent that the user can zoom in to.
A higher value means that the user can zoom in more.

<GraphExample id="max-zoom-showcase" config={{
	maxZoom: 4,
	enableZoom: true,
	enablePan: true,
}} baseConfig={{ ...focusSingleNode, scale: 1.1 }} sitemap={singleNodeSitemap} slug={"node/"}
	sliders={{
		"maxZoom": {
			"title": "maxZoom",
			"min": 0.1,
			"max": 5,
			"step": 0.1,
			"val": 4,
		}
	}}
/>


## Zoom Duration

The `zoomDuration` option determines the duration of the zoom animation in milliseconds.
This includes zooming in, zooming out and panning.

<GraphExample id="zoom-duration-showcase" config={{
	zoomDuration: 75,
	enableZoom: true,
	enablePan: true,
}} baseConfig={{ ...focusSingleNode, scale: 1.1 }} sitemap={singleNodeSitemap} slug={"node/"}
	sliders={{
		"zoomDuration": {
			"title": "zoomDuration",
			"min": 0,
			"max": 1000,
			"step": 25,
			"val": 75,
		}
	}}
/>

## Zoom Easer

The `zoomEase` option determines the easing function used for the zoom animation.
This includes zooming in, zooming out and panning.

<GraphExample id="zoom-easer-showcase" config={{
	zoomEase: "out_quad",
	enableZoom: true,
	enablePan: true,
}} baseConfig={{ ...focusSingleNode, scale: 1.1 }} sitemap={singleNodeSitemap} slug={"node/"}
	buttons={{
		"Linear": { config: { zoomEase: "linear" } },
		"Quadratic Ease In": { config: { zoomEase: "in_quad" } },
		"Quadratic Ease Out": { config: { zoomEase: "out_quad" } },
		"Quadratic Ease In Out": { config: { zoomEase: "in_out_quad" } },
	}}
/>


## Minimum Zoom Arrow Visibility

The `minZoomArrows` option determines at which zoom level the arrows on links should become visible.
A high value means that the arrows will only be visible when the user has zoomed in quite a bit.

<GraphExample id="min-zoom-arrows-showcase" config={{
	minZoomArrows: 0.8,
	enableZoom: true,
	enablePan: true,
	renderArrows: true,
}} baseConfig={frozenConfig} sitemap={threeNodeSitemap} slug={"node/"}
	sliders={{
		"minZoomArrows": {
			"title": "minZoomArrows",
			"min": 0.1,
			"max": 3,
			"step": 0.1,
			"val": 0.8,
		}
	}}
/>


## Scale Links

The `scaleLinks` option determines whether the width of links between nodes should be scaled when zooming.
This can be useful to keep the links consistent when zooming in or out.

<GraphExample id="scale-links-showcase" config={{
	scaleLinks: true,
	enableZoom: true,
	enablePan: true,
}} baseConfig={frozenConfig} sitemap={threeNodeSitemap} slug={"node/"}
	buttons={{
		"Scale Links": { config: { scaleLinks: true } },
		"Do Not Scale Links": { config: { scaleLinks: false } },
	}}
/>


## Scale Arrows

The `scaleArrows` option determines whether the size of arrowheads on links should be scaled when zooming.
This can be useful to keep the arrowheads consistent when zooming in or out.

<GraphExample id="scale-arrows-showcase" config={{
