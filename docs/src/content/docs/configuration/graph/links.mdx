Each node in the graph can be connected to other nodes via links.
Here, you can find how you can customize these links, including their width and whether arrows are rendered.

:::note[Note]
If you are looking for information on how to _color_ these links,
this can be done by changing its [CSS variables](/starlight-site-graph/configuration/css/variables#links).
:::

## Link Width

The `linkWidth` determines the width of the links between nodes.
The higher the value, the thicker the links.

<GraphExample id="link-width-showcase" config={{ graphConfig: {
	linkWidth: 1,
}}} baseConfig={zoomedOutConfig} sitemap={threeNodeSitemap} slug={"/node/"} sliders={{
	"graphConfig.linkWidth": {
		"title": "linkWidth",
		"min": 1,
		"max": 10,
		"step": 1,
		"val": 1,
	}
}} />


## Link Hover Width

The `linkHoverWidth` determines the width of the links between nodes when hovered over.
The higher the value, the thicker the links.

## Arrow Rendering

The `renderArrows` option toggles the rendering of arrows on the edges of the graph.
When arrows are rendered, they indicate the direction of the link between nodes.

<GraphExample id="render-arrows-showcase" config={{ graphConfig: {
	renderArrows: true,
}}} baseConfig={zoomedOutConfig} sitemap={threeNodeSitemap} slug={"/node/"}
	buttons={{
		"Hide Arrows": { config: { graphConfig: { renderArrows: false } } },
		"Render Arrows": { config: { graphConfig: { renderArrows: true } } },
	}}
/>


## Arrow Size

The `arrowSize` determines the size of the arrows on the links between nodes.
The higher the value, the larger the arrows.
Arrows between links are only rendered if the `renderArrows` option is enabled.

<GraphExample id="arrow-size-showcase" config={{ graphConfig: {
	renderArrows: true,
	arrowSize: 5,
}}} baseConfig={zoomedOutConfig} sitemap={threeNodeSitemap} slug={"/node/"} sliders={{
	"graphConfig.arrowSize": {
		"title": "arrowSize",
		"min": 1,
		"max": 20,
		"step": 1,
		"val": 10,
	}
}} />


## Arrow Angle

The `arrowAngle` determines the 'pointiness' of the arrows on the links between nodes.
The higher the value, the pointier the arrows.
Arrows between links are only rendered if the `renderArrows` option is enabled.

:::note
For this example, the value is expressed as a fraction of π.
The default value is π/6.
:::

<GraphExample id="arrow-angle-showcase" config={{ graphConfig: {
	renderArrows: true,
	arrowAngle: Math.PI / 6,
}}} baseConfig={zoomedOutConfig} sitemap={threeNodeSitemap} slug={"/node/"} sliders={{
	"graphConfig.arrowAngle": {
		"title": "arrowAngle",
		"min": 0,
		"max": Math.PI / 2,
		"step": Math.PI / 48,
		"val": Math.PI / 6,
	}
}} />
